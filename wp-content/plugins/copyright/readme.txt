== Description ==

adding the post link at the bottom of each post

== Installation ==

    * Download the "copyright" and unzip it.
    * Copy "copyright.php" to the wp-content/plugins directory.
    * Login to Wp-admin and go to Plugins and enable "Copyright Declaration".
